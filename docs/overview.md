# システム概要

ランチくじ（Lunch Roulette）は、Google マップに保存したお店リストから「今日のランチ」を
ランダムに提案してくれる、自分専用のシンプルなツール。

技術的な詳細は [ARCHITECTURE.md](../ARCHITECTURE.md) を、
使い方は [user-manual.md](user-manual.md) を参照。

## 目的・対象ユーザー

普段から Google マップに気になるお店を保存しているが、いざランチの時間に
なると「結局どこにするか」迷ってしまう、という悩みを解決するための
個人利用ツール。ジャンルや気分でお店を絞り込み、最後は「ランダムに選ぶ」
ボタンに決めてもらう。

## 主要機能

- **CSV 取り込み** — Google Takeout でエクスポートした保存リスト（CSV）を
  そのまま読み込める。ファイル名がエリア名として扱われる。
- **ジャンル自動推定** — 店名から「ラーメン」「カフェ」などのジャンルを
  自動で推定し、初期タグとして付与する（外部 AI は使わず、店名の
  キーワードマッチングのみで判定）。
- **タグ付け・編集** — ジャンル・気分タグは取り込み後に自由に編集できる。
- **絞り込み & ランダム提案** — エリア・ジャンル・気分で絞り込み、
  該当するお店の中から1件をランダムに提示する。
- **バックアップ / 復元** — 全データを JSON ファイルとして書き出し・
  読み込みできる。

## 重要な制約

- **データはこの端末のブラウザ内（localStorage）にのみ保存される。**
  サーバーには一切送信されない。ブラウザのデータを消去すると
  リストも失われるため、定期的な JSON バックアップを推奨する。
- 別の端末・ブラウザとはデータが同期されない（オフライン専用ツールのため）。
- AI（生成AI・外部API）は一切使用していない。ジャンル推定もすべて
  ローカルのルールベース処理。
