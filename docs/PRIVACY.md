# プライバシーポリシー（Privacy Policy）

最終更新日: 2026年6月26日

> 本書は一般的なひな型です。法的有効性を保証するものではなく、必要に応じて弁護士等の専門家にご相談ください。

本プライバシーポリシーは、本アプリ「ランチくじ（Lunch Roulette）」（以下「本アプリ」）における利用者の情報の取り扱いについて、本アプリの開発者（以下「当方」）が定めるものです。

## 1. 基本方針

本アプリは、**サーバー（バックエンド）を持たず、利用者の端末内で完結**して動作します。当方は、利用者が本アプリに取り込み・入力したデータ（店舗リスト・タグ等）を**収集・送信・閲覧・保存することはありません**。

## 2. 端末内に保存されるデータ

本アプリのすべてのデータは、利用者の端末のブラウザの **localStorage** にのみ保存されます。取り込んだ CSV の内容や付与したタグも端末内にとどまり、当方を含む第三者に自動的に送信されることはありません。

## 3. 外部へのデータ送信

本アプリは、利用者のデータを外部のサーバーや第三者サービスへ送信しません。データの書き出し／読み込み（JSON）はすべて利用者の端末内で行われます。

## 4. Cookie・アクセス解析

本アプリは、利用者を追跡する目的での Cookie やアクセス解析ツールを使用しません。

## 5. お問い合わせ

本ポリシーに関するお問い合わせは、本アプリのリポジトリの Issues 等を通じてお願いします。

## 6. 本ポリシーの変更

当方は、必要に応じて本ポリシーを変更することがあります。変更後の本ポリシーは、本アプリまたはリポジトリ上に掲示した時点で効力を生じます。
